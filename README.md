How to install
-------------
```
 1. [sudo] apt-get install nodejs
 2. [sudo] npm install bower -g
 3. [sudo] npm install forever -g
 4. git clone git@github.com:weishiji/order-monitor.git
 5. cd to path
 6. npm install
 7. cd to path/public
 8. bower install
 9. cd to path
 10. forever start ./bin/www
```
